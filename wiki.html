Example Article

From Wikipeek, the free encyclopedia
Example infobox
Type: Demo content
License: This page is for demonstration; no Wikipedia trademarks used.

Example Article is a sample page that mimics the layout of a typical Wikipedia article using only a single HTML file. It demonstrates a header, sidebar navigation, table of contents, an infobox, images, and standard article sections. You can deploy this file directly to GitHub Pages.

History

The layout on this page is inspired by the Vector skin used by MediaWiki (the software behind Wikipedia). The design has been recreated from scratch without using Wikipedia's trademarks, fonts, or assets.

Features

  • Single-file deployment (HTML + CSS)
  • Responsive layout (collapses nicely on mobile)
  • Infobox with an embedded image
  • Table of contents and semantic headings
  • Accessible colors and alt text
